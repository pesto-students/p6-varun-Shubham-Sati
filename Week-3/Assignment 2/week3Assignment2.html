Topic : Javascript Call , Bind And Apply

Please Look into browser console for the result

bind()
  • Bind return's the function.
  • With the bind() method an object can borrow a method from another object.
  • you can send your argument in the function seperately (seperated by comma and inside curly braces).
  • Sometime bind method is used to prevent loosing "this"
call()
  • Call don't return anything.
  • Call method is used to invoke/call a method with an owner object as an argument.
  • you can send an object and all your argument in the function seperately (seperated by comma and inside curly braces).
apply()
  • Apply don't return anything.
  • With the apply() method an object can borrow a method from another object.
  • you can send an object and all your argument inside an array to the apply function.
Note :
  • With call() you have to know the total arguments at compile time, but with apply() you can differ that decision at runtime.
